# Sales Performance Dashboard – Tableau
## Project Overview
This project presents an interactive Sales Performance Dashboard developed using Tableau. The dashboard provides insights into overall sales, profit, and quantity trends with Year-over-Year (YoY) comparison. It enables stakeholders to monitor performance, identify high-performing sub-categories, and analyze weekly sales patterns.

## Objectives
- Analyze total Sales, Profit, and Quantity
- Compare performance with previous year (YoY growth)
- Identify top-performing sub-categories
- Monitor weekly sales and profit trends
- Support data-driven business decisions

 ## Key Insights
- Total Sales reached $733K, showing a 20.36% Year-over-Year growth, indicating strong overall business expansion.
- Total Profit increased by 14.24% YoY, suggesting improved profitability but at a slightly lower rate than sales growth.
- Total Quantity sold grew by 26.83% YoY, showing increased customer demand and transaction volume.
- Sub-category analysis highlights that products like Phones and Chairs contribute significantly to overall sales performance.
- Weekly trends show fluctuations in profit despite consistent sales growth, indicating possible variations in cost structure or discount strategies.
